Native Windows GUI: The Guide

Welcome! Native Windows GUI (NWG) is a thin GUI toolkit built over the Windows WINAPI for rust. The current version is 0.2.0 BETA. The library is close to be production ready, but still lacks some important features and some useful controls and resources.

The NWG guide is split into chapters. After this introduction, follow these:

  • Getting started - Install NWG and create your first application.
  • Basics - The basics required to build an application with NWG.
  • Intermediate - Topic on how to structure a NWG application.
  • Custom - How to build custom events with NWG.
